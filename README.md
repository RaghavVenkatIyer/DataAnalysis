This is a repository that helps analyze 2 different JSON files with altitude, velocity, time data from 2 different rocket launches into readable graphs. 
The schema for the json files are attached in test1.json and test2.json. These 2 are test files with test/trial data. 
The python file, when run locally, will take the 2 json files as inputs and produce a full pdf with comparisons between the two launches(including graphs and tables), making the need for manual comparison unnecessary
